## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/ryan7998/deep-thoughts.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run start:dev
```

4. Seed the database (optional):

```bash
npm run seed
```
